Features

Plagiarism checker is a web-based application which finds out if a student has copied or used someone elseâ€™s work. The main aim of this project is to detect the percentage of similarity among the assignment that are submitted. The students nowadays tend to copy their assignments from their friends and the senior students and present them as their own work. This project helps to find out whether the submitted assignments are copied or not. If the assignment is found to be copied, then this project will help to detect the similar assignments and display the similarity in percentage. It mainly focuses on comparing assignments by using the longest common substring algorithm which is used to find the longest common substring in the document. This application consists of a database where all the assignment that are uploaded by the students are stored and further processed.
